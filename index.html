Meus Projetos

Alguns dos trabalhos que desenvolvi durante minha formação e estudos

Imagem do Projeto 1

Sistema de Gestão Escolar

Plataforma completa desenvolvida em Python para gerenciar alunos, professores, disciplinas e notas de uma instituição de ensino. O sistema conta com interface gráfica intuitiva criada com Tkinter, banco de dados SQLite para armazenamento das informações e funcionalidades como cadastro de usuários, lançamento de notas, geração de relatórios e controle de frequência. Este projeto me permitiu aplicar conceitos importantes de programação orientada a objetos, manipulação de banco de dados e desenvolvimento de interfaces desktop.

Python SQLite Tkinter POO
Imagem do Projeto 2

Site E-commerce

Loja virtual desenvolvida com JavaScript, HTML5 e CSS3, contando com carrinho de compras dinâmico, sistema de filtros por categoria e integração com API de pagamento. O projeto foi desenvolvido com foco em responsividade, garantindo uma experiência de usuário consistente em dispositivos móveis e desktop. Implementei funcionalidades como busca de produtos, avaliações de clientes, cálculo de frete e processo de checkout seguro. Esta experiência me proporcionou um entendimento prático sobre desenvolvimento front-end, consumo de APIs e princípios de UX/UI design.
